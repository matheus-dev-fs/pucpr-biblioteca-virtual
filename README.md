# Biblioteca Virtual (PUCPR) — Métodos de Pesquisa e Ordenação em Estruturas de Dados

Projeto desenvolvido para a disciplina **Métodos de Pesquisa e Ordenação em Estruturas de Dados** do curso de **Análise e Desenvolvimento de Sistemas** da **PUCPR**.

A aplicação simula uma **biblioteca virtual** com foco em praticar e consolidar conceitos da disciplina, especialmente **estruturas de dados**, **algoritmos de pesquisa** e **algoritmos de ordenação**, aplicados a um domínio simples (livros, autores, etc.).

## Objetivos

- Exercitar o uso de estruturas de dados no contexto de um sistema.
- Aplicar algoritmos de **pesquisa** e **ordenação** sobre coleções de dados.
- Comparar abordagens e entender impacto de complexidade e organização dos dados.

## Funcionalidades (visão geral)

- Cadastro e listagem de itens da biblioteca (ex.: livros).
- Consulta/pesquisa de registros por campos relevantes (ex.: título, código, autor).
- Ordenação dos registros conforme critérios definidos (ex.: título, id).
- Operações básicas de uma biblioteca virtual para testes e simulações.

## Estruturas e Algoritmos

Este projeto foi pensado para praticar os tópicos da disciplina, como:

### Pesquisa
Exemplos de abordagens normalmente utilizadas no contexto do trabalho:
- Pesquisa sequencial (linear)
- Pesquisa binária (quando aplicável, com dados ordenados)

### Ordenação
Exemplos de algoritmos tipicamente explorados:
- Bubble Sort
- Insertion Sort
- Selection Sort
- Merge Sort / Quick Sort (se implementados no projeto)


## Estrutura do projeto

A estrutura pode variar, mas em geral você encontrará:
- `src/` — código-fonte Java
- classes de modelo (ex.: Livro, Autor)
- classes/serviços para manipulação de coleções e regras
- implementações de pesquisa e ordenação (diretamente ou utilitários)
